# Trybe Futebol Club ⚽

Projeto de avaliação do módulo de back-end do da Trybe onde eu desenvolvi uma Api para uma aplicação front-end já preparada, onde minha Api o alimentava com as informações vindo do banco de dados.

# Stacks 📚

- React
- NodeJs
- Docker
- Express
- MySql
- Sequelize
- Sinon
- Mocha
- Chai
- Typescript
- JWT
